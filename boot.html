Bootstrap Component Reference Guide

1. Grid System

The foundation of Bootstrap layouts. It utilizes a 12-column system built with Flexbox. It relies on three primary building blocks: containers, rows, and columns.

col-sm-4
col-sm-4

Key Classes Explained:
  • .container: Centers content horizontally and provides responsive breakpoints.
  • .row: Wraps columns and corrects alignment using negative margins.
  • .col-sm-4: Defines a column span of 4 out of 12 columns on small screens and up.

2. Navbar

A responsive header navigation wrapper. It automatically collapses into a hamburger menu on mobile viewports and scales out to full horizontal menus on large screens.

4. Buttons & Groups

Custom styles for actions, links, forms, and control groups. Context classes shift semantic background sets natively.

Key Classes Explained:
  • .btn: Base class targeting interaction resets, inline heights, and pointer triggers.
  • .btn-success / .btn-primary: Applies explicit semantic palette matching configurations directly.

5. Modals

Pop-up dialog overlays to focus user actions. Requires JavaScript mechanics bundled inside Bootstrap distributions to toggle active states.

Component Overlay
This container works perfectly using Bootstrap's default transition properties.
Key Attributes Explained:
  • data-bs-toggle="modal": Custom JavaScript link tag hook that triggers overlay actions without target scripting logic.
  • data-bs-target="#id": Points explicitly to the container structure targeted for presentation alterations.

6. Alerts

Provides contextual feedback messages for typical user actions with optional dismiss controls.

Key Classes Explained:
  • .alert: Base layout padding controller providing structured box models.
  • .alert-dismissible: reserves structural right padding layouts to embed contextual close controls safely.

7. Form Controls

Provides styling modernizations over native standard element structures including inputs, selects, ranges, and custom checkboxes.

Key Classes Explained:
  • .form-label: Standard margin parameters aligning top header text structures cleanly.
  • .form-control: Maps interaction focus transitions, borders, shadows, and inline field properties across textual types.
